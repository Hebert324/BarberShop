## 📌 Sobre o Projeto

O **Barber Shop** é uma **landing page institucional** no formato **One Page**, responsiva e adaptável para diversos tipos de negócios — como barbearias, salões, estúdios, entre outros.

Este projeto contém as seguintes seções:
- **Header**
- **Menu de Navegação**
- **Home**
- **Sobre**
- **Serviços**
- **Depoimentos**
- **Contato**
- **Footer**
